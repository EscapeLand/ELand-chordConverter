# ELand-chordConverter

本项目已于2018-6-22停止维护. 项目新地址：[点我！](https://github.com/YuanWangZhe/E-N-TabConverter)

## 将一张图片形式的吉他谱转换成MusicXML，便于用Guitar Pro之类的打谱软件读取修改。

实际上更常用的应该是，拿来一张六线谱，扫出来拿Guitar Pro导入听个响 :D
